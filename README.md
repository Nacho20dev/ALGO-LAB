# ☕ ALGO::LAB | Java Engineering Platform

> Entorno dinámico para la documentación y visualización de algoritmos de alto nivel.

### 🚀 Características
- **Logic.Core:** Módulos interactivos para práctica de Vectores y Sorting.
- **Engineering Logs:** Bitácora técnica con soporte para bloques de código Java.
- **Persistence:** Sistema de edición y guardado en `localStorage`.
- **UI/UX:** Interfaz *Dark Mode* optimizada con Tailwind CSS y tipografía Plus Jakarta Sans.

### 🛠️ Stack
- HTML5 / CSS3 (Tailwind)
- JavaScript Moderno (ES6 Modules)
- Lucide Icons & Custom SVG Engineering Symbols
